# DOMination: CodeQuest JavaScript Assignment: DOMination

codequest-domination is a project meeting the requirements for the DOMination Assignment: Build a JavaScript Fundamentals Quiz. This activity involves building a JavaScript fundamentals quiz using HTML and JavaScript.

## Project URL

The project has been deployed to https://codequest-domination.netlify.app/.

Visit the URL for easy viewing of the project. If desired, the project can be cloned to your local machine using the instructions in the "Installation" instructions.

## Installation

1. Run git clone https://github.com/Heather-Rutherford/DOMination.git
2. Navigate to and open the index.html in a web browser.

or

Visit Netlify. The project is deployed to: https://codequest-domination.netlify.app/

## Goals of the Project

DOMination: CodeQuest JavaScript Assignment: DOMination has an index.html file that is the JavaScript fundamentals quiz. The design was translated from the example video for the assignment: https://codingtemple.disco.co/p/software-engineering-1whu3/curriculum/overview?filterContentLabelId=Q29udGVudExhYmVsOjQ0NzUy&u=0f8a07e8-bd3b-4e07-98e8-a0edd3a44604.

## What I did

1. Analyze the video: Look at layout, colors, typography, and components.
2. Stage the components: Placed containers in the index.html page.
3. Convert to Code: Used HTML, and CSS.
4. Showcase Your Skills: Used JavaScript to build the functionality of the quiz in the page.
